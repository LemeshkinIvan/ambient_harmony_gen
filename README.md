# ambient_harmony_gen

Эта аппка для генерации случайной последовательности аккордов в **C мажоре** и сохранения её в формате **MIDI**.  
Последовательность аккордов и их надстройки выбираются случайным образом из файла `const.py` (пока захардкожено).  

Программа **не синтезирует звук**, а только создаёт midi-файлы, которые можно будет редактировать в любой DAW.

---

## Особенности

- Генерация случайной гармонии
- Сохранение результатов в MIDI
- Быстрое создание фоновых аккордов для музыки
- Лёгкость последующего редактирования сгенерированных файлов

---

## Установка

1. Клонируйте репозиторий:

```bash
git clone https://github.com/LemeshkinIvan/ambient-harmony-generator.git
cd ambient-harmony-generator

python main.py

```
